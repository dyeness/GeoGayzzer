# 🌍 GeoGAYZZER

> Многопользовательская игра в угадывание местоположения по уличным панорамам. Смотри на Mapillary-панораму — угадай, где ты находишься. Побеждает тот, кто ближе к цели.

---

## ✨ Возможности

| Фича | Описание |
|------|----------|
| 🎯 **Одиночная игра** | 5 раундов с автоматически найденными панорамами |
| 👥 **Мультиплеер** | До 10 игроков в одной комнате через Radmin VPN или локалку |
| 🗡️ **Кража очков** | Угадал ближе соперника в пределах 50 км — забираешь 20% его очков |
| 📊 **Рейтинг ЭЛО** | Шахматная система ELO: растёт при победах, падает при поражениях |
| 🎖️ **30 достижений** | За точность, кражи, победы, опыт и многое другое |
| ⭐ **Уровни и XP** | Набирай опыт за каждый раунд и матч, повышай уровень |
| 💎 **Престиж** | Каждые 55 уровней — новая ступень престижа `[N💎]` |
| 👤 **Профили** | Рекорды, достижения, история последней игры |
| 🗺️ **Mapillary** | Настоящие уличные панорамы со всего мира |
| 🔭 **Предзагрузка** | Кэш из 100+ заранее найденных рабочих панорам |

## 🚀 Быстрый старт

### Требования

- **Node.js** v18+ — [nodejs.org](https://nodejs.org)
- **npm** (входит в Node.js)
- Современный браузер (Chrome, Firefox, Edge)

### Установка

```powershell
# 1. Перейди в папку проекта
cd E:\Gits\GeoGAYZZER

# 2. Установи зависимости (один раз)
npm install

# 3. Запусти сервер
node server/index.js
```

При успешном запуске увидишь:

```

─────────────────────────────────────────────
  Radmin VPN: http://26.201.251.196:3000
  Local:      http://localhost:3000
─────────────────────────────────────────────
```

Открой **http://localhost:3000** в браузере — и вперёд.

Поделись с друзьями адресом `http://26.201.251.196:3000` (или тем IP, что показывает сервер в разделе `Radmin VPN`).

## 🎮 Как играть

### Одиночная игра

1. Зарегистрируйся или войди
2. Нажми **«Одиночная игра»**
3. Изучи панораму — покрути камеру, зум
4. Кликни на мини-карте, где по-твоему это место
5. Нажми **«УГАДАТЬ»**
6. Узнай расстояние и счёт
7. 5 раундов → итоговый результат

**Максимум: 25 000 очков** (5 000 × 5 раундов)

### Мультиплеер

**Хост:**
1. Нажми **«Создать комнату»** → получи 6-значный код
2. Выбери цвет своего маркера (24 на выбор)
3. Поделись кодом комнаты с друзьями
4. Когда все зашли — нажми **«Начать игру»**

**Игрок:**
1. Нажми **«Присоединиться»**
2. Выбери комнату из списка или введи код
3. Жди старта от хоста

Все игроки видят одну панораму и угадывают одновременно. После каждого раунда — таблица с результатами и маркерами всех угадываний на карте.

## 🗡️ Механика кражи

Если двое угадали **в пределах 50 км** друг от друга, более точный игрок забирает **20% очков** менее точного. Это отображается на экране результатов раунда красным значком.

## 📊 Система рейтинга

### XP и уровни

| Событие | XP |
|---------|----|
| Очки за раунд | очки ÷ 8 |
| 1-е место в раунде | +200 |
| 2-е место в раунде | +100 |
| 3-е место в раунде | +60 |
| Точность 90%+ | +50 |
| Точность 99%+ | +100 |
| Угадал < 1 км | +75 |
| Угадал < 100 м | +200 |
| Кража очков | +30 |
| 1-е место в матче | +600 |
| Участие в матче | +50 |
| Безупречный матч (все раунды 90%+) | +200 |

Формула уровня: **уровень N требует ⌊200 × N^1.5⌋ XP**

### ELO

Рейтинг считается по **шахматной формуле** (K=32) только в многопользовательских матчах. Стартовый рейтинг — **1000**. Минимум — **100**.

## 🐛 Частые проблемы

**`EADDRINUSE: address already in use :3000`**
```powershell
Get-Process -Name node | Stop-Process -Force
node server/index.js
```

**`Cannot find module 'express'`**
```powershell
npm install
```

**Панорама не загружается**
- Проверь интернет-соединение
- Убедись, что Mapillary-токен актуален в `config/api.js`
- Используй предзагрузку (меню → «Предзагрузить панорамы»), чтобы накопить кэш

**Мультиплеер не работает**
- Все игроки должны быть в одной Radmin VPN
- Проверь IP в выводе сервера (раздел `Radmin VPN`)
- Порт по умолчанию: `3000`
