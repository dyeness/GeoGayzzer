# 🌍 GeoGuessr — Multiplayer Edition

Локальная версия GeoGuessr с поддержкой Street View (Mapillary), Leaflet картами и мультиплеем через Radmin VPN.

---

## 🚀 Быстрый старт

### Требования
- **Node.js** v18+ ([https://nodejs.org](https://nodejs.org))
- **npm** (идёт с Node.js)
- Браузер с поддержкой ES6 (Chrome, Firefox, Safari, Edge)
- **Mapillary токен** (уже добавлен в проект)

### Установка и запуск (Windows)

#### 1️⃣ Откройте PowerShell/cmd и перейдите в папку проекта:
```powershell
cd E:\Gits\GeoGuessr
```

#### 2️⃣ Установите зависимости (один раз):
```powershell
npm install
```

#### 3️⃣ Запустите сервер:
```powershell
node server/index.js
```

Вы должны увидеть:
```

─────────────────────────────────────────────
  Radmin VPN: http://26.201.251.196:3000
  Ethernet: http://10.0.0.3:3000
  Local:   http://localhost:3000
─────────────────────────────────────────────
```

#### 4️⃣ Откройте браузер:
- **Локально:** http://localhost:3000
- **По Radmin VPN:** http://26.201.251.196:3000 (поделись этим IP с друзьями!)

## ⚠️ Ошибка: "address already in use 0.0.0.0:3000"

Это значит, что порт 3000 уже занят другим процессом.

### Решение 1️⃣: Убить процесс на порту 3000

**Windows (PowerShell):**
```powershell
Stop-Process -Name node -Force -ErrorAction SilentlyContinue
```

Затем снова запустите:
```powershell
node server/index.js
```

### Решение 2️⃣: Использовать другой порт

**Отредактируйте server/index.js:**

Найдите строку:
```javascript
const PORT = process.env.PORT || 3000;
```

Измените на:
```javascript
const PORT = process.env.PORT || 5000;  // или любой другой порт
```

Затем откройте http://localhost:5000

## 🎮 Как играть

### Одиночная игра (Solo)
1. Введи никнейм
2. Нажми "Одиночная игра"
3. Смотри Mapillary панораму
4. Кликни на мини-карте (справа внизу) где по-твоему это место
5. Нажми "УГАДАТЬ"
6. Видишь результат с расстоянием и очками
7. 5 раундов → итоговый счёт

**Максимум: 25 000 очков (5000 × 5 раундов)**

### Мультиплеер (Multiplayer через Radmin VPN)

#### Для хоста:
1. Введи никнейм
2. Нажми "Создать комнату"
3. Получишь **код комнаты** (например: `ABC123`)
4. Поделись кодом и своим IP Radmin с друзьями
5. Когда все подключились → нажми "Начать игру"
6. Все видят одну панораму, угадывают одновременно
7. После всех → видишь результаты раунда (кто где угадал)
8. Хост нажимает "Следующий раунд"

#### Для подключающегося игрока:
1. Введи никнейм
2. Нажми "Присоединиться"
3. Введи **IP:порт хоста** (например: `26.201.251.196:3000`)
4. Введи **код комнаты** хоста
5. Нажми "Подключиться"
6. Жди, пока хост начнёт игру

## 🎯 Формула подсчёта очков

```
Расстояние 0 км       → 5000 очков
Расстояние 100 км     → 4881 очков
Расстояние 1000 км    → 3069 очков

```

Формула: `5000 × e^(-расстояние_км / 2000)`

## 🐛 Решение проблем

### Проблема: "Cannot find module 'express'"
**Решение:** Ты не установил зависимости. Запусти:
```powershell
npm install
```

### Проблема: Панорама не загружается
**Решение:** 
- Проверь интернет
- Убедись, что Mapillary токен актуален (в config/api.js)
- Не все места имеют Street View — игра автоматически ищет близлежащие изображения

### Проблема: Мультиплеер не работает
**Решение:**
- Оба игрока должны быть подключены в одну Radmin VPN
- Проверь IP адрес хоста (серверный вывод при запуске показывает все доступные IP)
- Убедись, что правильно введён порт (по умолчанию 3000)

### Проблема: Страница загружается, но не работает
**Решение:**
- Нажми F12 → Console
- Смотри ошибки в консоли браузера
- Чаще всего это проблема с браузером (обнови страницу Ctrl+F5)
