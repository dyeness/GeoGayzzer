# Идеи для улучшения GeoGuessr

## Геймплей
- **Режим "Горячий картофель"** — угадавший первым передаёт пинг команде-сопернику
- **Режим "Командная игра"** — 2 команды, очки суммируются по команде
- **Лимит времени на раунд** — таймер с обратным отсчётом (30–120 сек.), настраивается хостом
- **Зоны раундов** — хост выбирает регион (Европа, Азия, Россия и т.д.)
- **Стрик-бонус** — бонус за N точных угадываний подряд (< 500 км)
- **Панорама-шторм** — каждые 10 секунд изображение немного ухудшается (blur/sepia)

## Рейтинг и прогрессия
- **Сезонный рейтинг** — ELO сбрасывается каждые 30 дней, наград за топ-позиции
- **Лига-система** — Bronze/Silver/Gold/Platinum/Diamond лиги с плейсментом
- **Матч-история с картами** — запись всех угадываний с отображением на мини-карте
- **Достижения за стрики** — "10 раундов без пропуска угадывания"
- **Персональные рекорды** — лучший счёт в разные регионы

## Социальные функции
- **Реакции в чате** — быстрые эмодзи (👍 😂 🔥 💀) по клику без ввода текста
- **Профиль публичный** — страница `/profile/:nickname` доступная без входа
- **Инвайт-ссылка** — кнопка "Скопировать ссылку на комнату" в лобби (уже есть "копировать код" — расширить до полного URL)
- **Мут игрока** — локальный мут сообщений чата от конкретного игрока
- **Уведомление о входе** — системное сообщение в чате при заходе/выходе игрока

## UI/UX
- **Тёмный/светлый режим** — переключатель темы в настройках
- **Мобильная оптимизация** — тач-управление картой, responsive лейаут
- **Анимация результатов** — поочерёдный reveal результатов раунда с задержкой
- **Spectator mode** — наблюдение за игрой без участия
- **Полноэкранный режим** — кнопка F11 / fullscreen API для панорамы
- **Zoom на мини-карте** — колёсико мыши и кнопки ±

## Технические улучшения
- **Persistent rooms** — сохранение состояния комнаты при перезапуске сервера
- **Anti-cheat базовый** — проверка что guess не отправлен раньше загрузки панорамы
- **Rate-limiting чата** — не более 5 сообщений в 10 секунд
- **Websocket heartbeat** — пинг/понг для обнаружения "зависших" соединений
- **Панорама CDN** — кеш прогрева панорам заранее для популярных локаций
- **Admin dashboard** — веб-страница `/admin` для управления комнатами и профилями

## Панорамы и карты
- **Метаданные панорам** — показ страны/города после угадывания
- **Категории локаций** — городские/природа/сельские, хост выбирает микс
- **Загрузка кастомных координат** — хост загружает CSV с lat/lng
- **Street View fallback** — если Mapillary не даёт панораму, попробовать другой источник
- **Популярные локации** — отдельная коллекция "известных мест"
